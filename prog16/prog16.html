Usability Test

Link to Collector App

Results of User Test

The User

The user is a grad student. She spends 4-5 hours a week on the internet. She usually uses a phone and computer.

Initial Thoughts

The user thinks the page looks barren and needs graphics.

Task 1: Pull up a Lechonk card

The user took 28.18 seconds to complete. The task was described as simple with nothing being notably confusing. The user wishes there was more pictures or thumbnails for the cards instead of ID.

Task 2: Navigate to the About Page

The user took 48.43 seconds. They got lost and used the navbar to go to Home because they assumed About and Home were the same. She said she was thrown off by API because she doesn't know what that is. The task could have been worded better to go to the API About page.

Task 3: Navigate to the API Official Documentation Link

The user took 7.95 seconds. She found the link from the previous task.

Improvements

  1. Add back button
  2. Add more card details
  3. Change placement of About
  4. Add thumbails during search
  5. Add help text under search bar
